# 格界 — 编译 & 安装说明

## 项目结构

```
格界/
├── Makefile
├── control
├── Sources/
│   ├── main.mm                    # App 入口
│   ├── AppDelegate.mm/h           # 应用生命周期，启动画面
│   ├── GJTheme.mm/h               # 全局主题（颜色/字体/组件）
│   ├── APIClient.mm/h             # 网络层（登录/获取任务/提交任务）
│   ├── LoginViewController.mm/h  # 登录界面（动态渐变背景）
│   ├── TaskViewController.mm/h   # 任务主界面（完整工作流）
│   ├── TaskDetailViewController.mm/h  # 任务详情页
│   ├── FloatBubbleView.mm/h       # 可拖动悬浮气泡
│   └── FloatWindowManager.mm/h   # 悬浮窗管理（独立 UIWindow）
└── 格界-编译说明.md
```

---

## 功能流程

```
启动 → 检查 Token
       ├─ 有 Token → 任务主界面（TaskViewController）
       └─ 无 Token → 登录界面（LoginViewController）

登录界面：
  输入账号密码 → POST /api/user/login → 获取 Token → 保存到 NSUserDefaults

任务主界面（4步工作流）：
  Step 1  [获取任务]    GET /api/task/take → 获得 taskUrl
  Step 2  [跳转商品]   openURL → 打开 Shopee/Apple App 商品页
  Step 3  [获取详情]   解析 Shopee API → 获取商品名/价格/评分
  Step 4  [提交任务]   POST /api/task/submit/v2 → 计数 +1 → 悬浮窗脉冲动画

悬浮窗：
  - 常驻在所有 App 之上（UIWindowLevel = Alert + 1000）
  - 拖动吸附边缘
  - 徽标显示完成数量
  - 点击执行当前步骤
  - 长按显示快捷菜单
```

## 编译步骤（WSL Ubuntu）

### 1. 确保 Theos 已安装
```bash
bash setup_wsl.sh  # 在 MyTweak 目录中
```

### 2. 编译格界 App
```bash
cd /mnt/c/Users/Administrator/Desktop/ios/格界
make package
```

### 3. 生成的 deb 包
```
packages/com.gejie.app_1.0.0_iphoneos-arm.deb
```

## 安装到越狱 iPhone

### 方式一：SSH 直接安装
```bash
# 设置手机 IP（确保与电脑同一 WiFi）
export THEOS_DEVICE_IP=192.168.1.xxx
make install
```

### 方式二：手动安装
```bash
# 传输 deb 到手机
scp packages/*.deb root@192.168.1.xxx:/var/root/

# SSH 到手机
ssh root@192.168.1.xxx

# 安装
dpkg -i /var/root/com.gejie.app_1.0.0_iphoneos-arm.deb

# 刷新图标缓存
uicache -p /Applications/GeJie.app
```

### 方式三：Filza 安装
把 deb 文件传到手机，用 **Filza** 文件管理器点击安装。

## 接口配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 服务器地址 | `https://eqwofaygdsjko.uk` | 可在悬浮窗长按菜单修改 |
| Token 存储 | NSUserDefaults `GJ_TOKEN` | 自动持久化，重启后免登录 |

### API 接口
| 功能 | 方法 | 路径 |
|------|------|------|
| 登录 | POST | `/api/user/login` |
| 获取任务 | GET | `/api/task/take` |
| 提交任务 | POST | `/api/task/submit/v2` |

## 悬浮窗说明

- **点击**：执行当前工作流下一步
- **长按**：弹出快捷菜单（获取任务/配置/退出登录）
- **拖动**：自由移动，松手后自动吸附到屏幕左/右边缘
- **徽标**：显示当前会话累计完成数量
- **脉冲**：每次任务成功提交后触发绿色光晕动画
